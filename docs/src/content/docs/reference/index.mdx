# API Reference

This section contains detailed API documentation for all bfra.me Works packages.

## Package APIs

- **[@bfra.me/create](/packages/create)**: Project creation API and CLI
- **[@bfra.me/eslint-config](/packages/eslint-config)**: ESLint configuration API
- **[@bfra.me/prettier-config](/packages/prettier-config)**: Prettier configuration API
- **[@bfra.me/semantic-release](/packages/semantic-release)**: Release configuration API
- **[@bfra.me/tsconfig](/packages/tsconfig)**: TypeScript configuration reference

## Configuration Schemas

### TypeScript Configuration

```typescript
interface TSConfig {
  extends: string
  compilerOptions: CompilerOptions
  include: string[]
  exclude: string[]
}
```

### ESLint Configuration

```typescript
interface ESLintConfig {
  files: string[]
  rules: Record<string, RuleConfiguration>
  plugins: string[]
}
```

### Prettier Configuration

```typescript
interface PrettierConfig {
  printWidth: number
  tabWidth: number
  useTabs: boolean
  semi: boolean
  singleQuote: boolean
  trailingComma: 'none' | 'es5' | 'all'
  bracketSpacing: boolean
  arrowParens: 'avoid' | 'always'
}
```

For detailed API documentation, please refer to the individual package documentation pages.
