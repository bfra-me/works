## What is bfra.me Works?

bfra.me Works is a collection of tools and shared configurations designed to streamline modern JavaScript and TypeScript development. It provides consistent, opinionated configurations for ESLint, Prettier, TypeScript, and more, along with utilities for creating new projects and managing releases.

## Quick Start

Get started with bfra.me Works in minutes:

```bash
# Create a new project
npx @bfra.me/create my-project

# Or install individual packages
pnpm add -D @bfra.me/eslint-config @bfra.me/prettier-config @bfra.me/tsconfig
```

## Philosophy

bfra.me Works embraces these core principles:

- **Consistency**: Shared configurations ensure consistent code style across projects
- **Opinionated**: Sensible defaults that work out of the box
- **Modular**: Use only what you need, packages work independently
- **Modern**: Built for contemporary JavaScript and TypeScript development
- **Extensible**: Easy to customize and extend for your specific needs
